درباره پروژه

این یک قالب خیلی ساده است تا بتونی رویش کامیت بزنی، برنچ بسازی و با گیت تمرین کنی. از این پروژه می‌توانی برای یادگیری اصول اولیه Git مانند commit، branch، merge و push استفاده کنی.

دستورات Git برای تمرین:

  • git init - ایجاد ریپازیتوری جدید
  • git add . - اضافه کردن تغییرات
  • git commit -m "پیام" - ثبت تغییرات
  • git branch نام-برنچ - ایجاد برنچ جدید
  • git checkout نام-برنچ - جابجایی بین برنچ‌ها

ویژگی‌ها

  • ساختار ساده و قابل فهم
  • فقط یک فایل HTML و یک فایل CSS
  • مناسب برای تمرین گیت و گیت‌هاب
  • طراحی واکنش‌گرا برای موبایل و دسکتاپ
  • دارای افکت‌های تعاملی و انیمیشن
  • قابلیت تغییر رنگ با JavaScript
